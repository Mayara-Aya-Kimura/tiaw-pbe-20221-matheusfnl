Geometry Dash


Geometry Dash é um jogo de ritmo desenvolvido pelo programador sueco Robert Topala. Atualmente, o jogo dispõe de 22 mapas distintos, sendo que cada um deles possui músicas únicas e configurações exponencialmente desafiadoras. Assim como diversos jogos de desse gênero, cada mapa desafia alguma habilidade específica do jogador, podendo variar entre coordenação motora, paciência, memória muscular, entre outros.

★★★★☆


Requisitos Mínimos

SO: Windows XP
Processador: 2.0+ GHz
Memória: 512 MB de RAM
Placa de vídeo: OpenGL 2.0 support
Armazenamento: 100 MB de espaço disponível
Placa de som: XAudio2 / DirectSound compatible


Requisitos Recomendados

SO: Windows 7 ou versões mais recentes
Processador: Intel® Core™ i5 ou superior
Memória: 2 GB de RAM
Placa de vídeo: NVIDIA GeForce® GTX 970 / AMD Radeon™ R9 290 ou superior (para VR)
Armazenamento: 50 MB de espaço disponível
Placa de som: XAudio2 / DirectSound compatible
